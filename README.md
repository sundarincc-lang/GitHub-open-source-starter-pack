# GitHub Open Source Starter Pack 🎉

Welcome to the **GitHub Open Source Starter Pack**!  
This repository is designed to help beginners understand Git, GitHub, and how to start contributing to open-source projects with confidence.

---

## 📖 Contents
- **Getting Started**: Introduction and  the process of contributing to open source using GitHub.  
- **Cheat Sheets**: Quick references for essential Git commands  
- **Workflow Diagrams**: Visuals explaining Git & GitHub collaboration  
- **Examples**: Sample repositories and branching strategies  
- **Contribution Guide**: Step-by-step instructions on forking, cloning, branching, and opening pull requests  
- **Resources**: Links to tutorials, documentation, and community best practices
